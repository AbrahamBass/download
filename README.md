# Download YouTube

Download YouTube es una aplicación que te permite descargar videos de YouTube de la manera más sencilla y eficiente posible. Utiliza la biblioteca `pytube` para gestionar las descargas y proporciona una interfaz rápida y amigable para que los usuarios obtengan copias locales de sus videos favoritos de YouTube.

## Instalación

Para utilizar Download YouTube en tu entorno local, sigue estos pasos:

### Clonar el Repositorio

Primero, clona este repositorio en tu máquina local utilizando Git:

```bash
git clone https://github.com/yourusername/yourrepository.git
```

### Instalación de Dependencias

Después, navega hasta el directorio del proyecto e instala las dependencias necesarias usando pip:

```bash
cd yourrepositoryv
pip install
```

### Ejecutar la Aplicación

Una vez instaladas las dependencias, inicia el servidor de Download YouTube con el siguiente comando:

```bash
uvicorn main:app --reload
```
